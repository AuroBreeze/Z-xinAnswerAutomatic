# Z-xinAnswerAutomatic

## 使用python进行对知新网站的内容获取自动检测答案，但仅支持单选和判断。

### 功能
- 登录知新网站
- 自动获取题目
- 自动获取答案
- 自动检查AI答案正确率

### 注意

在 `https://bailian.console.aliyun.com/#/home` 中获取api_key
